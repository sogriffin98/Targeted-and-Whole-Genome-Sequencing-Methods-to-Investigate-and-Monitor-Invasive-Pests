The first data chapter of my PhD project focuses on *Vespa velutina* (yellow legged or Asian hornet). The data chapter is split into 2 parts:
* Population Genetics
* Kinship Analysis

I have included the code and input files used for both parts in separate folders listed below relating to the analysis that they are for.
