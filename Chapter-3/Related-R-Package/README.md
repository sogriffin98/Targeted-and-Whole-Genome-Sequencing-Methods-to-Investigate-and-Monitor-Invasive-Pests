# Creating the input files for Related R Package
* Using PGDSpider (https://software.bioinformatics.unibe.ch/pgdspider/ and https://doi.org/10.1093/bioinformatics/btr642) convert the filtered VCF file into a STRUCTURE file
* Make sure to save the output as a .txt file for input in the R code
* An example output is the UK_ALL_Relatedness_convert.txt
